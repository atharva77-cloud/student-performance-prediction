# Student Performance Prediction using Machine Learning

## Overview
This project predicts student performance using Machine Learning models.

## Dataset
Students Performance Dataset

## Technologies Used
- Python
- Pandas
- NumPy
- Matplotlib
- Scikit-Learn
- Jupyter Notebook

## Models Used
1. Logistic Regression
2. Decision Tree
3. Random Forest

## Results
| Model | Accuracy |
|---------|---------|
| Logistic Regression | 89.5% |
| Decision Tree | 83% |
| Random Forest | 87% |

Best Model: Logistic Regression
